FlavorDome is a mobile-first web app that guides you through a totally unique, objective taste test experience. This was my final group project for Momentum, created with Ryan Dowd and Stephen Kwan, which built upon everything we'd learned at Momentum and added new tools and frameworks (Vue, Vuex, websockets with Django Channels) not introduced to us in class to unlock FlavorDome's full social potential.

Mystery Word

code

As an introduction to Python, I was tasked with creating a game of Mystery Word. The program imports a list of over 200,000 words, allows the user to choose a difficulty level, prompts user guesses, reveals correctly-guessed letters, stores and counts down incorrect guesses, and resolves the game appropriately when the user succeeds or fails in guessing the word.
